Git for beginners

  • git init - this will initialize a got repository
  • git status - you will see the status of the repository (untracked files etc.)
  • git add . - will add all the files of your project folder to the tracking logic of git
  • git commit -m - tells git to commit the changes ( adds the code as a version of the code) and gives the commit a name
  • git branch - view branches
  • git log - branch log

    • Git does not create copies of each commit that you create. When you add "git add ." git has a look at your different files in the code and sees if you change something and if you commit that in the end these chanes are saved, but git is not creating copies of the code again and again. If you want to revert all unstaged changes and go back to your last commit:

  • git checkout -- .

    • ADDING NEW FEATURES/BRANCHES

  • git checkout -b new-feature(name of the new branch)
  • Add new file
  • git add .
  • git commit -m"css added"
  • git master - go to the master branch
  • git merge new-feature

    • DELETING BRANCH

  • git branch -D new-feature
