Github for beginners

  • git remote add origin (URL from github)
  • git remote - you can see what you have added and what you already had before
  • git remote -v
  • git push -u origin master
  • git branch -r - "origin/master", means that you have a connection between your local master branch and the master branch in your remote repository
  • git clone (URL from github) - clones the entire repository (for somebody else to use for an example)
  • git fetch
  • git pull
